## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Git (optional, for local development)

### Running Locally

1. Clone the repository:
```bash
git clone https://github.com/phuongddx/ios-news.git
cd ios-news
```

2. Open `index.html` in your browser:
```bash
open index.html
# or
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## 📝 Adding New Articles

To add a new article:

1. Open `index.html`
2. Add a new `<article>` element inside `.news-grid`:
```html
<article class="news-card">
    <div class="card-header">
        <div class="card-number">06</div>
        <div class="card-actions">
            <button class="card-action" title="Bookmark">🔖</button>
            <button class="card-action" title="Share">🔗</button>
        </div>
    </div>
    <h2><a href="YOUR_URL" target="_blank" rel="noopener">YOUR TITLE</a></h2>
    <div class="news-meta">
        <span class="tag swift">CATEGORY</span>
        <span>📅 DATE</span>
    </div>
    <div class="news-content">
        YOUR DESCRIPTION
    </div>
    <a href="YOUR_URL" target="_blank" rel="noopener" class="read-more">Read more</a>
</article>
```

3. Update the RSS feed in `rss.xml`
4. Commit and push:
```bash
git add .
git commit -m "Add new article: YOUR TITLE"
git push
```
